# ResiAll_Moni
Desenvolvimento de um aplicativo de segurança, projetado para proteger dados do usuário em cenários de furto ou coação. Ao inserir uma senha de pânico, o sistema ativa um modo de segurança que apresenta um ambiente de fachada, enquanto executa em segundo plano um protocolo de backup assíncrono para a nuvem e a exclusão segura de dados críticos.
